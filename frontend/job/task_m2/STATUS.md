## 1. 状态枚举

| 状态 | 含义 |
| --- | --- |
| `未开始` | 尚未派发或无人认领 |
| `进行中` | Agent 已开工，代码未验收 |
| `已完成` | 本任务 AC/DoD 已通过 |
| `已合并` | 已合入团队约定的集成分支 |
| `阻塞` | 因依赖或环境问题无法继续 |

**下游依赖以 `已合并` 为准**；单分支开发时 `已完成` 可视同 `已合并`。
